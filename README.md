# TraumaTeam

O projeto busca integrar 4 diferentes sistemas de gerenciamento de bancos de dados para catalogar informações sobre diferentes entidades. De modo que cada SGBD possuí é utilizado de modo a explorar sua maior vantagem em nível de projeto.

# O projeto
No distânte ano de 2077, a TraumaTeam se destaca como a maior corporação médica de Night City, oferecendo planos médicos aos maiores nomes da cidade. Porém um problema para a corporação é o grande mercado negro de implantes, cada vez mais comum entre os mais pobres.
Você foi contratado como Trilha-rede para desenvolver um software capaz de rastrear instalações de implantes e a atividade de médicos em Night City. Armazenando informações sobre Cyberwares, Medicânicos, Clientes e as Corporações fornecedoras dos implantes.

# Os SGBDS

# MongoDB
Será resposável por armazenar informações densas e voláteis, principalmente a respeito dos Cyberwares, que recebem atualizações e novos lançamentos constantemente.

# Neo4j
Será responsável por armazenar as relações entre os Cyberwares e as demais entidades, como a qual corporação o Cyberware pertênte, quais medicânicos o vendem e por quais preços e quais indivíduos já o adquiriram.

# Cassandra
Será responsável por armazenar informações altamente relevantes e que exigam alta disponibilidade, como dados de medicânicos.
